* Compile 64 bit versions only, as freeglut.dll is a 64 bit dll.
* The development environment: Windows 10; Visual Studio 2019; OpenGL2.0.

* Third party libraries
  •	freeglut, an openGL library, available at:
    https://sourceforge.net/projects/freeglut/
  •	Imgui, a C++ lightweight cross-platform GUI library, available at:
    https://github.com/ocornut/imgui
  •	stb_image, an easy-to-use image library, available at:
    https://github.com/nothings/stb
